前言

blog中需要使用一些随机的图片和句子，于是在网络上找了一些，把一些感觉挺有价值的整理放到这里。

随机文字类别的API：

  • 一言
    一言指一句话，提供一句话服务，包含动画、漫画、游戏、小说等七个分类，这个应该是相当有名了。

  • 今日诗词
    根据时间、地点、天气、事件智能推荐，好像很腻害。每次返回一句诗词

  • 古诗词·一言API
    今日诗词的旧版

  • https://www.liutianyou.com/260.html

  • 随机网名
    看起来都好中二

  • 乐评
    随机获取网易云音乐热评
